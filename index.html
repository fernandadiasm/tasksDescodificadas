Tarefas do curso Descodificadas

Trilha 1

Tarefa 8: Seletores

Descrição: criar um projeto onde o site fique apenas com quadrados coloridos.

Tarefa 9: Box model

Descrição: criar um projeto que reproduza a imagem de referência usando as configurações para Box Model.

Tarefa 10: Ada Lovelace

Descrição: Com o aprendizado da mentoria, faça um site que contenha o conteúdo conforme mostrado na imagem de modelo.

Tarefa 12 - Grid

Descrição: Criar um álbum com 10 imagens de tamanhos diferentes, utilizando o Grid.

Tarefa 13 - Animações

Descrição: Coloque uma animação para fazer o coração pulsar (ficar batendo).

Tarefa 14 - Introdução ao JS

Descrição: Criar um arquivo HTML e JS que tenha um script onde será declarada uma variável que receberá seu nome e que o apresente em uma mensagem de alerta automaticamente na tela.

Tarefa 15 - Variáveis

Descrição: Desenvolva uma página HTML que, ao ser carregada, exibe um alerta informando que acabou de ser carregada. Declare duas variáveis coloque um valor em cada uma e faça: soma, divisão, multiplicação, comparação para ver se os valores são iguais

Tarefa: Desenvolver um site

Descrição: projeto desenvolvido com HTML, CSS e JavaScript.

Trilha 2

Tarefa 7: Carrossel

Descrição: criar um carrossel utilizando apenas HTML e CSS.

Tarefa 8 - Grid

Descrição: Criar um álbum com 10 imagens de tamanhos diferentes, utilizando o Grid.

Tarefa 10 - Funções

Descrição: Faça um projeto que contenha uma função JavaScript em que o usuário informe 5 números e o programa calcule a média aritmética desses números e mostre o resultado.
